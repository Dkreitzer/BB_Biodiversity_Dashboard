What is the Belly Button Biodiversity Dataset?

In 2011 the Rob Dunn Lab launched an investigation to study the microbs living in human navels. For more specific information on the dataset, how the data was collected, and the project history; please visit their site here.

For this dashboard, we used both Belly Button Data Sets 1 & 2. Using a Jupyter Notebook, the CSV files provided by Rob Dunn Lab were converted and combined into a sqlite file.

To view the dashboard locally, it needs to be launched using a Flask Application. You can download the entire app, supporting dataset and instructions on my Github page.
